# casino

基于vue实现**纯前端**在线赌场（线上娱乐城博彩平台）

(我不会写后端)



预览：[casino](https://xzgame888.netlify.app/)



包含游戏：

- 龙虎斗
- 轮盘
- 21点
- 骰子
- 百家乐
- 捕鱼
- 持续更新主流游戏（含CS开箱玩法，区块链玩法）...

实现功能：

- 注册登录
- 充值提现
- 投注记录
- 投注统计

登陆注册以及投注记录等各种信息使用本地会话存储

## 项目部署

```
pnpm install
```

### 开发环境

```
pnpm run serve
```

### 打包

```
pnpm run build
```
